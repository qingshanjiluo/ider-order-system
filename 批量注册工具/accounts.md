## 📋 防封建议

### 同IP多号检测机制（[`auth.js`](源代码/server/routes/auth.js:98)）
- 服务器实现: [`auth.js`](源代码/server/routes/auth.js:106) 的 `getAccountsByLoginIp()`
- 触发条件: 同一 IP 登录 ≥ 3 个账号时触发
- 处罚规则: 按等级排序，保留等级最高的 3 个，其余封禁
- 封禁时长: 1次=3天 → 2次=7天 → 3次+=永久

### 已实施的防护措施
1. ✅ **极致IP伪装 v2.0** - 每个账号独立真实中国运营商 IP（电信/联通/移动 31个网段）
2. ✅ **每账号独立 machine_id** - 5种不同格式轮换
3. ✅ **浏览器指纹仿真** - Sec-CH-UA、Accept-Language、User-Agent 轮换
4. ✅ **操作随机延迟** - 账号间 3-6s，每5个暂停 20-40s
5. ✅ **智能分段** - 注册操作每 3 个暂停 20-40 秒
6. ✅ **完整登录流程** - 注册→创建角色→绑定邀请码→装备技能→铁剑→吐纳法→切换地图
